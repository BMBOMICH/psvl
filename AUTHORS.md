## Inspirations

PSVL draws structural inspiration from several existing licenses,
without copying their text:

- **Business Source License (BUSL) 1.1** — for the source-available
  conceptual model
- **Server Side Public License (SSPL)** — for service-provider obligations
- **Elastic License 2.0** — for limited-grant patterns
- **PolyForm licenses** — for use-case-restricted grants
- **Contributor Covenant** — for the Code of Conduct

PSVL is otherwise an independent work.
