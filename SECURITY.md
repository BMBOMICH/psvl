# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 1.0 | ✅ |

## Reporting a Vulnerability

If you discover a security issue with the PSVL template text
(for example, a clause that creates an unintended legal loophole
or a conflict between sections), please report it privately.

### How to Report

1. Go to the [**Security and quality**](https://github.com/BMBOMICH/PSVL/security) tab of this repository
2. Click **"Report a vulnerability"**
3. Describe the issue clearly, including the specific clause
   numbers affected

### What to Expect

- Acknowledgment within **48 hours**
- Assessment and response within **14 days**
- Credit in the changelog if your report leads to a fix

### ❌ Do Not

- Open a public issue for security-related concerns
- Disclose the vulnerability publicly before it is addressed

## Scope

This policy covers the PSVL template text itself. If you are
using PSVL in your own project and experience a security incident,
your own completed license governs — not this repository.
