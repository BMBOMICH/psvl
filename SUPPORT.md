# Getting Help with PSVL

Thank you for using PSVL. Here is where to go for help.

## Quick Routing

| What you want to do | Where to go |
|---|---|
| Ask a question about how PSVL works | [**GitHub Discussions — Q&A**](https://github.com/BMBOMICH/psvl/discussions/categories/q-a) |
| Suggest a clause idea before formally proposing it | [**GitHub Discussions — Clause Ideas**](https://github.com/BMBOMICH/psvl/discussions/categories/clause-ideas) |
| General discussion about source-visible licensing | [**GitHub Discussions — General**](https://github.com/BMBOMICH/psvl/discussions/categories/general) |
| Report a typo, formatting issue, or bug | [**Open an Issue**](https://github.com/BMBOMICH/psvl/issues/new/choose) |
| Report a clause conflict or legal loophole | [**Open an Issue**](https://github.com/BMBOMICH/psvl/issues/new/choose) |
| Propose a new clause | [**Open an Issue**](https://github.com/BMBOMICH/psvl/issues/new/choose) |
| Submit a code or text fix | [**Open a Pull Request**](https://github.com/BMBOMICH/psvl/pulls) |
| Report a security or legal-loophole issue privately | [**Report a vulnerability**](https://github.com/BMBOMICH/psvl/security/advisories/new) |
| Report a Code of Conduct violation | See [**CODE_OF_CONDUCT.md**](CODE_OF_CONDUCT.md) |

## What PSVL Support Is Not

The maintainer is a single individual, not a law firm. Please understand:

- **PSVL is not legal advice.** See [**DISCLAIMER.md**](DISCLAIMER.md).
- **Response times vary.** Best-effort, no SLA.
- **No commercial support tier.** PSVL is provided as a free template.
- **Jurisdiction-specific legal questions** should go to a licensed attorney
  in your country, not to this project.

## Before You Ask

1. Check the [**README.md**](README.md) — it covers most common questions.
2. Search [**existing issues**](https://github.com/BMBOMICH/psvl/issues).
3. Search [**existing discussions**](https://github.com/BMBOMICH/psvl/discussions).
4. Read the relevant section of [**PSVL.txt**](PSVL.txt).

## Response Expectations

- **Discussions:** When the maintainer is available.
- **Issues:** Triaged when the maintainer is available.
- **Security reports:** Acknowledged as quickly as practicable.
- **Pull Requests:** Reviewed when the maintainer is available.

PSVL is maintained by one person in their spare time. Patience is appreciated.
